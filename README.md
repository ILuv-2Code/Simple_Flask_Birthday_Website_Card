## Birthday Webpage
A simple Flask-based birthday webpage that displays a personalized message, plays background music on interaction, and lets the user download a gift by tapping a birthday cake GIF.

**Features**
- Personalized birthday greeting using Flask templates
- Mobile-friendly, responsive layout
- Background music that plays on first tap/click (mobile-safe)
- Clickable birthday cake that downloads a gift (gift.zip)
- Minimal, dark-themed design

**Tech Stack**
- Python (Flask)
- HTML / CSS
- JavaScript (for audio interaction)

**How to Run**
1) Just install Flask (pip install flask)
2) Run app.py (python app.py)

**Customization**
\n Edit app.py to change the name and message:
- name = "Name Here"
- text = "Birthday Message Here"
Edit index.html to change...
- 8-bit-happy-bday.mp3 with any audio file
- gift.zip with your own downloadable gift
